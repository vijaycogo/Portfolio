Solar Powered Smart Home Design with IoT|| IIT Patna link

In this paper, we design a solar powered smart home with wireless sensor network which deals with smart energy management, smart health care, smart ventilation, along with smart people management. We have used Bluetooth, ZigBee, principal of Maximum power point tracker.

Solar Power Smart Home Design with IOT Under Dr. R.K.Behera || IIT Patna

Developed a Home Automation System based on IoT using Wi-fi based microcontroller that can be controlled remotely. we design automated door opening , security alert , smart ventilation , and smart health care using iot and arduino.

Automatic Number Plate Recognition || IIT Patna

Implemented an ANPR model that recognizes the license plate of vehicles in real time. Trained custom YOLOv4 model on our own custom dataset for license plate detection. Detected license plate was further processed for character recognition using easy OCR. Achieved an accuracy of 91% on single lane test dataset and 85% on multi lane test dataset.

Smart Railway System Using IoT Under Dr. R.K.Behera || IIT Patna

With the objective to develop automatic track monitoring , freight saftey, automatic train ticketing (using microcontrollers and combination of sensor like ir,gps,ultrasonic,strain gauge etc ) that can reduce the cost of indian railway by 35%.
